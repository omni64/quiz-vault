# Celebrity real (birth) names

A durable, evergreen chestnut format — unlike tabloid gossip, birth names don't go stale.

| Stage name | Birth name |
|---|---|
| **Freddie Mercury** | Farrokh Bulsara |
| **Elton John** | Reginald Kenneth Dwight |
| **Marilyn Monroe** | Norma Jeane Mortenson (later Baker) |
| **Bob Dylan** | Robert Allen Zimmerman |
| **Vin Diesel** | Mark Sinclair |
| **Lady Gaga** | Stefani Joanne Angelina Germanotta |
| **Cary Grant** | Archibald Alec Leach |
| **John Wayne** | Marion Robert Morrison |

> [!tip] Hook
> "Real name of X" is one of the most-recycled celebrity clue shapes across every quiz format — a small, fixed list is worth memorising cold rather than guessing.

## Cluster
- **Famous sibling acts**: the **Baldwin brothers** (Alec, Daniel, William, Stephen); the **Gyllenhaals** (Jake and Maggie); the **Kardashian-Jenners** (half-sisters spanning two marriages of Kris Jenner).
- **Stage-name pattern**: many performers adopt a **mother's maiden name or a grandparent's name** as a new surname (a common trivia "why did they change it" angle) — verify specifics per performer before stating as fact.
- Distinguish **legal name changes** (Cary Grant, John Wayne — used exclusively, even legally, off-screen) from **stage names only** (many musicians keep their birth name for legal/personal use).

## Mnemonic
Say the ordinary birth name first, then "...but you know them as..." — the surprise of the mundane real name is what makes it stick.
