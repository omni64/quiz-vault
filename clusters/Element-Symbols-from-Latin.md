# Element symbols that don't match their English name

The single most-recycled chemistry chestnut on any quiz circuit: a handful of elements have symbols from their **Latin** (or German) names, not their modern English ones.

| Symbol | Element | Origin name |
|---|---|---|
| **Na** | Sodium | *Natrium* |
| **K** | Potassium | *Kalium* |
| **Fe** | Iron | *Ferrum* |
| **Cu** | Copper | *Cuprum* |
| **Ag** | Silver | *Argentum* |
| **Au** | Gold | *Aurum* |
| **Sn** | Tin | *Stannum* |
| **Pb** | Lead | *Plumbum* |
| **Hg** | Mercury | *Hydrargyrum* ("liquid silver") |
| **Sb** | Antimony | *Stibium* |
| **W** | Tungsten | *Wolfram* (German, not Latin — the odd one out) |

> [!tip] Hook
> Clued as "what element has the symbol Na/Fe/Pb/etc." or the reverse. **Tungsten/W** is the trap answer for "which symbol isn't from Latin" — it's German.

## Cluster
- The periodic table currently has **118 confirmed elements**; the most recent four (**Nihonium, Moscovium, Tennessine, Oganesson** — 113, 115, 117, 118) were formally named in **2016**.
- **Absolute zero** is **0 K = −273.15°C = −459.67°F** — the temperature at which the Kelvin scale starts.
- The **speed of light** in a vacuum is **299,792,458 m/s**, exactly (it defines the metre since 1983).
- **Mercury (Hg)** is the only metal that's liquid at room temperature; **bromine** is the only non-metal liquid at room temperature.

## Mnemonic
Say the symbol, then its Latin root, aloud: "**Na** – Natrium, **Fe** – Ferrum, **Cu** – Cuprum, **Ag** – Argentum, **Au** – Aurum, **Sn** – Stannum, **Pb** – Plumbum." The symbol *is* the mnemonic once you know the Latin word.
