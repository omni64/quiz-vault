# 64. Gadolinium (Gd)

**Discovered by:** Jean Charles Galissard de Marignac, 1880, Geneva, Switzerland (named after Johan Gadolin)

## Unique properties / facts
- One of the best neutron absorbers known — used in reactor control/shielding
- Gadolinium chelates are the standard MRI contrast agent
- Ferromagnetic near room temperature — unusual among elements

## Common compounds
- Gd-DTPA — MRI contrast agent
- Gd2O3
