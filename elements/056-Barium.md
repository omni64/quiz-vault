# 56. Barium (Ba)

**Discovered by:** Recognized by Carl Wilhelm Scheele, 1774, Sweden; isolated by Humphry Davy, 1808, England

## Unique properties / facts
- Barium sulfate ("barium meal") used as an X-ray contrast agent — insoluble, so non-toxic despite barium metal's toxicity
- Barium salts produce green fireworks
- Name from Greek "barys" (heavy)

## Common compounds
- BaSO4 — X-ray contrast medium
- BaCO3 — rodenticide
