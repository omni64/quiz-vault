# 100. Fermium (Fm)

**Discovered by:** Albert Ghiorso et al., 1952 (announced 1955), Berkeley/Argonne/Los Alamos, USA — also from "Ivy Mike" debris

## Unique properties / facts
- Named after physicist Enrico Fermi
- Heaviest element producible via neutron-capture in reactors — everything heavier must be made atom-by-atom in accelerators
- Never produced beyond trace/tracer amounts

## Common compounds
- None well-characterized — only tracer-level solution chemistry studied
