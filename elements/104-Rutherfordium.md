# 104. Rutherfordium (Rf)

**Discovered by:** Jointly credited — JINR Dubna, USSR (1964) and Lawrence Berkeley Laboratory, USA (1969); IUPAC's 1997 resolution adopted Berkeley's proposed name

## Unique properties / facts
- First transactinide element
- Named after Ernest Rutherford
- Single-atom gas-chromatography experiments show RfCl4 is volatile, confirming Rf as the expected heavier homolog of Ti/Zr/Hf

## Common compounds
- RfCl4, RfBr4 — studied only via single-atom volatility experiments
