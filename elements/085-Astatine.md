# 85. Astatine (At)

**Discovered by:** Dale Corson, Kenneth MacKenzie, Emilio Segrè, 1940, Berkeley, California, USA

## Unique properties / facts
- Rarest naturally occurring element on Earth (under 1 gram in the crust at any time)
- So short-lived no visible/weighable sample has ever existed
- Name from Greek "astatos" — "unstable"

## Common compounds
- None characterized — too short-lived for isolation
