# 55. Caesium (Cs)

**Discovered by:** Robert Bunsen & Gustav Kirchhoff, 1860, Germany — first element discovered by spectroscopy

## Unique properties / facts
- Name from Latin "caesius" (sky blue), for its spectral lines
- Defines the SI second via the caesium atomic clock
- Melts just above room temperature (~28°C)

## Common compounds
- CsCl — caesium chloride
