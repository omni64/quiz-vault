# 4. Beryllium (Be)

**Discovered by:** Louis Nicolas Vauquelin, 1798, France; isolated 1828

## Unique properties / facts
- Named after beryl (emerald's mineral)
- Transparent to X-rays — used in X-ray tube windows
- Toxic — causes berylliosis

## Common compounds
- BeO — beryllium oxide (ceramics)
- Be3Al2(SiO3)6 — beryl/emerald
