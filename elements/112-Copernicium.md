# 112. Copernicium (Cn)

**Discovered by:** Sigurd Hofmann, Victor Ninov et al., 1996, GSI Darmstadt, Germany

## Unique properties / facts
- Named after astronomer Nicolaus Copernicus
- Gas-phase experiments suggest it behaves like an extremely volatile "noble liquid/gas," chemically more inert than mercury
- Predicted to be gaseous at room temperature, unlike mercury directly above it

## Common compounds
- None characterized in bulk — only single-atom adsorption experiments
