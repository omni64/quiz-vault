# 27. Cobalt (Co)

**Discovered by:** Georg Brandt, 1735, Sweden

## Unique properties / facts
- Name from German "kobold" (goblin) — miners blamed the ore for poisoning workers
- "Cobalt blue" is a historic glass/ceramic pigment
- Vitamin B12 contains cobalt; Co-60 used in cancer radiotherapy

## Common compounds
- CoCl2 — color-changing humidity indicator
- Cobalt blue glass (cobalt aluminate)
