# 60. Neodymium (Nd)

**Discovered by:** Carl Auer von Welsbach, 1885, Vienna, Austria

## Unique properties / facts
- Name means "new twin" (neos + didymos)
- Core of the strongest commercial permanent magnets (NdFeB)
- Neodymium glass used in some lasers (Nd:YAG)

## Common compounds
- Nd2O3 — glass colorant
- Nd2Fe14B — NdFeB magnets
