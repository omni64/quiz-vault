# 84. Polonium (Po)

**Discovered by:** Marie & Pierre Curie, 1898, Paris, France — named after Poland, Marie Curie's homeland

## Unique properties / facts
- First element named to make a political point (Poland's lack of independence)
- Used to poison ex-KGB agent Alexander Litvinenko in London, 2006
- Used in anti-static brushes and spacecraft heat sources (RTGs)

## Common compounds
- PoO2
- PoBe — neutron source
