# 76. Osmium (Os)

**Discovered by:** Smithson Tennant, 1803, London, England

## Unique properties / facts
- Densest naturally occurring element
- Named from Greek "osme" (smell) due to volatile OsO4's odor
- OsO4 used as a staining agent in electron microscopy

## Common compounds
- OsO4 — osmium tetroxide
- OsCl3
