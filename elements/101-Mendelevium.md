# 101. Mendelevium (Md)

**Discovered by:** Albert Ghiorso, Glenn Seaborg, Gregory Choppin, Bernard Harvey, Stanley Thompson, 1955, Berkeley, California, USA

## Unique properties / facts
- Named after Dmitri Mendeleev, creator of the periodic table
- First element identified one atom at a time — only ~17 atoms made in the discovery run
- Longest-lived isotope has a half-life of only about 51 days

## Common compounds
- None characterized — only single-atom ion-exchange chemistry studied
